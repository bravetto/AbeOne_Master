## üöÄ QUICK START

### Convert Markdown to CDF

```bash
python3 scripts/cdf_converter.py input.md output.cdf
```

### Convert CDF to Other Formats

```bash
# To Markdown
python3 scripts/cdf_parser.py input.cdf markdown output.md

# To HTML
python3 scripts/cdf_parser.py input.cdf html output.html

# To JSON
python3 scripts/cdf_parser.py input.cdf json output.json
```

### Index Documents with Genius Scores

```bash
# Index single file
python3 scripts/cdf_genius_indexer.py document.cdf

# Index entire directory
python3 scripts/cdf_genius_indexer.py ./documents/ index.json
```

# BiasGuard Chrome Extension

**Protect yourself from AI bias in real-time conversations**

BiasGuard is a powerful Chrome extension that automatically detects and alerts you to potential bias in AI-generated content across websites. Whether you're using ChatGPT, Claude, or any other AI platform, BiasGuard helps ensure you receive fair, unbiased responses.

## Why Use BiasGuard?

AI systems can sometimes produce biased or unfair content that affects decision-making, hiring, and daily interactions. BiasGuard acts as your personal bias detector, helping you:

- **Identify biased language** in AI responses before it influences your decisions
- **Get real-time alerts** when potentially biased content is detected
- **Make informed choices** with unbiased information
- **Protect yourself and others** from discriminatory AI outputs

## Installation

1. Visit the [BiasGuard Chrome Extension page](https://chrome.google.com/webstore)
2. Click "Add to Chrome"
3. Confirm installation when prompted
4. The BiasGuard icon will appear in your Chrome toolbar

## Getting Started

### Step 1: Install the Extension
1. Click the BiasGuard icon in your Chrome toolbar
2. The extension will open and show the main interface

### Step 2: Enable Bias Detection
1. Toggle "Enable Bias Detection" to ON
2. Choose which websites you want BiasGuard to monitor
3. Start browsing - BiasGuard will automatically detect bias in AI conversations

### Step 3: Understanding Alerts
When BiasGuard detects potential bias:
- You'll see a notification in your browser
- The extension popup will show details about the detected bias
- You can choose to ignore, report, or learn more about the bias type

## How It Works

BiasGuard uses advanced AI technology to analyze conversations in real-time:

1. **Content Analysis**: Scans text for patterns that may indicate bias
2. **Context Understanding**: Considers the conversation context for accurate detection
3. **Bias Classification**: Identifies specific types of bias (gender, racial, age, etc.)
4. **User Notification**: Alerts you when potential bias is detected
5. **Learning**: Improves detection accuracy over time

## Troubleshooting

### Extension Not Working?
- Check that bias detection is enabled in the extension popup
- Try refreshing the webpage
- Restart your browser if issues persist

### Not Receiving Alerts?
- Verify the website is supported by BiasGuard
- Check that the bias detection toggle is ON
- Ensure you have an active internet connection

### Performance Issues?
- Close unnecessary browser tabs
- Clear your browser cache
- Update to the latest version of BiasGuard
- Contact support if problems continue

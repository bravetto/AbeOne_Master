# Bravetto AI Chat Widget for Shopify

This extension adds an AI-powered chat widget to your Shopify storefront, allowing customers to get instant help and answers to their questions.

## Installation

The widget is automatically installed when you install the Bravetto AI app from the Shopify App Store.

## Setup Instructions

### Method 1: Enable Globally (Recommended)

1. From your Shopify admin, go to **Online Store > Themes**
2. Click **Customize** on your current theme
3. In the theme editor, click **App embeds** in the left sidebar
4. Find **Bravetto AI Chat** and toggle it ON
5. Click on the app embed to configure:
   - Chat title and welcome message
   - Widget position (bottom left or right)
   - Colors to match your brand
   - Pages to hide the widget on (optional)
6. Click **Save**

### Method 2: Add to Specific Pages

1. In the theme editor, navigate to the page where you want the chat
2. Click **Add section** or **Add block** (depending on your theme)
3. Search for **Bravetto AI Chat**
4. Configure the widget settings
5. Click **Save**

## Configuration Options

### Basic Settings
- **Enable Chat Widget**: Turn the widget on/off
- **Chat Title**: The title shown in the chat header
- **Welcome Message**: First message customers see
- **Input Placeholder**: Text shown in the input field

### Appearance
- **Widget Position**: Bottom left or bottom right
- **Button Color**: Color of the chat toggle button
- **Header Color**: Color of the chat header
- **User Message Color**: Color of customer messages

### Advanced Settings
- **API URL**: Backend endpoint (pre-configured)
- **System Prompt ID**: AI behavior configuration
- **Hide on Pages**: Comma-separated list of pages to exclude (e.g., "cart,checkout")

## Troubleshooting

### Widget Not Showing
1. Ensure the app embed is enabled in your theme
2. Check if the widget is hidden on the current page
3. Clear your browser cache
4. Make sure the Bravetto AI app is installed and active

### Chat Not Responding
1. Verify your internet connection
2. Check the browser console for errors
3. Ensure the API URL is correct in settings
4. Contact support if the issue persists

### Styling Issues
1. The widget automatically adapts to your theme
2. Use the color settings to match your brand
3. The widget has a high z-index to appear above other elements
4. Test on different devices to ensure proper display

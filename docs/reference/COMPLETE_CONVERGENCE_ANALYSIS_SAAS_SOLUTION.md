### 2.1 Cognitive Bias Patterns (180+ Identified)

#### **Critical Biases for SaaS Detection:**

1. **Anchoring Bias** (40-60% of numerical tasks)
   - First option disproportionately selected
   - Initial prompt framing dominates reasoning

2. **Confirmation Bias** (30-50% accuracy reduction)
   - Seeks information confirming hypothesis
   - Contradictory evidence underweighted

3. **Framing Effect** (13%+ systematic error)
   - Binary vs. continuous format bias
   - Question wording changes answers

4. **Status Quo/Omission Bias** (60-75% biased against action)
   - Systematically biased against taking action
   - Default to "no change" recommendations

5. **Order Bias** (30-40% answer changes)
   - First/last options favored
   - Position affects selection probability

**Detection Requirements:**
- Pattern recognition for 180+ cognitive biases

- Debiasing prompts
- Format normalization

class UnifiedQualityScore:

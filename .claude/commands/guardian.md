# `/guardian` Command

Manage guardian agents and their states.

## Usage

```
/guardian [action] [name]
```

## Actions

- `status` - Check guardian status
- `activate` - Activate a guardian
- `amplify` - Amplify guardian capabilities
- `validate` - Validate guardian state
- `list` - List all guardians

## Examples

```
/guardian status AEYON
/guardian activate JOHHN
/guardian list
```

## Usage

```
/disk-onboard [options]
```

## Options

- `--onboard` — Onboard discovered disks
- `--disk <name>` — Onboard specific disk
- `--discover` — Only discover disks (default)

## Examples

```bash
/disk-onboard                    # Discover all disks
/disk-onboard --onboard          # Discover and onboard all disks
/disk-onboard --disk Elements    # Onboard specific disk
```

## Execution

**Command Handler:** `scripts/disk-epistemic-onboarding.py`

When this command is invoked, execute:
```bash
python3 scripts/disk-epistemic-onboarding.py [options]
```
